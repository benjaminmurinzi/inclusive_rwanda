## üì∞ Managing News

### Adding a News Article

1. **Click "Manage News"** in sidebar
2. **Fill in the form:**
   - **Article Title:** Catchy, descriptive title
   - **Author Name:** Your name or "Admin"
   - **Article Content:** Full article text (minimum 50 characters)
   - **Featured Image (Optional):** Upload article image
   - **Published:** Check to make article visible on website

3. **Click "Add Article"**

### Editing a News Article

1. **Click "Edit"** next to article
2. Update content
3. **Click "Update Article"**

### Publishing/Unpublishing

- **Checked box:** Article is visible on website
- **Unchecked box:** Article is draft (not visible)

### News Image Guidelines
- **Format:** JPG, PNG, GIF, WEBP
- **Size:** Maximum 5MB
- **Dimensions:** Recommended 1200x630px (landscape)
- **Content:** Relevant to article

## ‚úÖ Best Practices

### Content Management

**DO:**
- ‚úÖ Use clear, descriptive titles
- ‚úÖ Check spelling and grammar
- ‚úÖ Optimize images before uploading
- ‚úÖ Test how content looks on website
- ‚úÖ Keep content up to date
- ‚úÖ Use consistent formatting

**DON'T:**
- ‚ùå Upload very large images (>5MB)
- ‚ùå Leave fields blank
- ‚ùå Use all caps in titles
- ‚ùå Copy content from other websites
- ‚ùå Delete content without backup

### Image Optimization

**Before uploading:**
1. Resize images to appropriate dimensions
2. Compress images (use tinyjpg.com)
3. Use descriptive filenames
4. Check image orientation

**Recommended Tools:**
- TinyJPG (https://tinyjpg.com) - Compress images
- Canva (https://canva.com) - Resize and edit
- IrfanView (Windows) - Batch processing

### Security

**Password Security:**
- Change default password immediately
- Use strong passwords (12+ characters)
- Don't share credentials
- Log out when finished

**Content Security:**
- Back up content regularly
- Don't upload suspicious files
- Review uploads before publishing
- Keep software updated

### Regular Maintenance

**Daily:**
- Check for new contact messages
- Respond to inquiries

**Weekly:**
- Review and update news
- Check email logs
- Test website functionality

**Monthly:**
- Backup database
- Backup uploaded files
- Review analytics
- Update old content

## üÜò Common Issues & Solutions

### Issue: Can't Login

**Solutions:**
1. Check username and password
2. Clear browser cookies
3. Try different browser
4. Contact technical support

### Issue: Image Won't Upload

**Solutions:**
1. Check file size (<5MB)
2. Verify file format (JPG, PNG, GIF, WEBP)
3. Try different image
4. Check internet connection

### Issue: Changes Not Showing

**Solutions:**
1. Clear browser cache (Ctrl + F5)
2. Check if published
3. Wait a few minutes
4. Check database was updated

### Issue: Email Not Received

**Solutions:**
1. Check spam folder
2. Verify email address is correct
3. Check email logs
4. Test email system
